Il s’agit d’une application API RESTful développée en PHP en utilisant le framework Symfony.
Cette application est principalement chargée de la gestion de la sécurité et de la mise à disposition des API d’authentification et d’autorisation des utilisateurs de l’application.

Le framework Symfony est utilisé pour assurer :

La sécurisation des accès aux différentes ressources de l’API

La gestion de l’authentification des utilisateurs (connexion, déconnexion, vérification des identités)

La gestion des autorisations et des rôles utilisateurs

La protection contre les vulnérabilités courantes (CSRF, accès non autorisés, etc.)

L’API respecte les principes REST, permettant une communication standardisée et sécurisée avec les applications clientes (web ou mobile).
